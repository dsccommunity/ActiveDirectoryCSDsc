TOPIC

SHORT DESCRIPTION
    DSC resources for installing, uninstalling and configuring Certificate
    Services components in Windows Server.

LONG DESCRIPTION
    This module contains DSC resources for installing, uninstalling and configuring
    Certificate Services components in Windows Server.

EXAMPLES
    PS C:\> Get-DscResource -Module ActiveDirectoryCSDsc
